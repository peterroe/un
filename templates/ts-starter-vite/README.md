## ts-starter-vite

A typescript template with `vite`

## Try it now!

```bash
$ pnpm create un
# or
$ pnpm create un [my-project] -t ts-starter-vite
```

Init git hook(optional)

```shell
$ pnpm git-hook-init
```

## Development

Run `src/index.ts`

```shell
$ pnpm start
```

## Test

```bash
$ pnpm test
```

## Build

Build your lib through `vite`

```shell
$ pnpm build
```

## Publish

Publish your library to npmï¼š

```shell
$ pnpm release
$ npm publish
```

## Different from ts-starter

Bundling with `vite`, browser friendly.
