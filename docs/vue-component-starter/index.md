# vue-component-starter

Quickly set up a vue component project

## Installation

```bash
$ pnpm create un
? Project name: › un-project ✔
? Select a framework: ›
    cli-starter
    docs-starter
    monorepo-starter
    ts-starter
    ts-starter-vite
❯   vue-component-starter
✔ Select a framework: › vue-component-starter
✔ Download template...

Done. Now run:

  cd un-project
  pnpm install
```

or

```bash 
$ pnpm create un un-project -t vue-component-starter
```
