# ts-starter

快递搭建一个 ts 项目

## 安装

```bash
$ pnpm create un
? Project name: › un-project ✔
? Select a framework: ›
    cli-starter
    docs-starter
    monorepo-starter
❯   ts-starter
    ts-starter-vite
    vue-component-starter
✔ Select a framework: › ts-starter
✔ Download template...

Done. Now run:

  cd un-project
  pnpm install
```

or

```bash 
$ pnpm create un un-project -t ts-starter
```
